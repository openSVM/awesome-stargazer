# WASM TOOLS (2 repositories)

perky/ZigGraphics: A graphics library written in Zig that uses opengl. Also cross compiles to WebAssembly and has the correct JS and HTML files without needing Emscripten. - https://github.com/perky/ZigGraphics
ptitSeb/gl4es: GL4ES is a OpenGL 2.1/1.5 to GL ES 2.0/1.1 translation library, with support for Pandora, ODroid, OrangePI, CHIP, Raspberry PI, Android, Emscripten and AmigaOS4. - https://github.com/ptitSeb/gl4es
