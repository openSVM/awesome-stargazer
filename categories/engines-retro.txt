# ENGINES RETRO (3 repositories)

pk-industries/traces: Playdate Horror game being written in Love2D, then refactored to be playable on the Playdate.  - https://github.com/pk-industries/traces
OpenRA/OpenRA: Open Source real-time strategy game engine for early Westwood games such as Command & Conquer: Red Alert written in C# using SDL and OpenGL. Runs on Windows, Linux, *BSD and Mac OS X. - https://github.com/OpenRA/OpenRA
zipxing/rust_pixel: Rust 2d game engine, ideal for developing mini-games&app, support terminal, sdl, web modes. - https://github.com/zipxing/rust_pixel
